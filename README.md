# escritoryInventory
aplicacion de escritorio para manejo de inventario y personas que lo utilizan, creado en usando GO y Fyne


## Descripción

Esta es una aplicación de inventario desarrollada en Go utilizando la biblioteca de interfaz de usuario Fyne. La aplicación permite a los usuarios gestionar un inventario y controlar quién puede acceder y modificar el inventario.

## Características

- **Gestión de Inventario**: Los usuarios pueden agregar, eliminar y modificar elementos en el inventario.
- **Control de Acceso**: La aplicación tiene roles de usuario para administradores y usuarios generales. Los administradores tienen permisos completos para modificar el inventario, mientras que los usuarios generales tienen permisos limitados.

## Instalación

Para instalar la aplicación, sigue estos pasos:

1. Asegúrate de tener Go instalado en tu sistema.
2. Clona este repositorio en tu sistema local.
3. Navega hasta el directorio del proyecto y ejecuta `go build` para compilar la aplicación.
4. Ejecuta el archivo binario generado para iniciar la aplicación.

## Uso

Para utilizar la aplicación, inicia sesión con tus credenciales de usuario. Si eres un administrador, podrás agregar, eliminar y modificar elementos en el inventario. Si eres un usuario general, podrás ver el inventario pero no modificarlo.
